安裝GIT

設定基本資料


----------------------------------資料夾設定倉儲-----------------------------
要整理的資料夾按右鍵，GIT BRACH


----------------------------------使用倉儲-----------------------------
初始化倉儲（repository）
git init

砍掉倉儲（repository）
rm -rf .git

檔案加入版控追蹤
git add .
git add [file]

檔案取消版控追蹤
git rm --cached <file_name>


提交（commit）新建版本
git commit -m 'dev1_11505051240'

取消 commit
git reset HEAD^ 
git reset HEAD^ --mixed


版本歷史紀錄
git log



查詢所在分支（branch）的版控狀態
git status

----------------------------------分支branch-----------------------------

建立分支
git branch 'dev1'

刪除分支
git branch -d 'dev1'

強制刪除分支
git branch -D 'dev1'

查詢分支
git branch

切換分支
git checkout 'dev1'

回復到未修改前的狀態
git checkout <branch-name> <file-name>
git checkout -- .

建立分支+切換分支（branch）
git branch -b 'dev1'

合併分支（branch）
git merge <branch-name>

----------------------------- 上傳至 GitHub -----------------------------
$ git remote add origin <遠端 repo 位址>
$ git push -u origin master


將本地資料推送（push）至遠端倉儲（repository）
git push <remote_name> <branch_name>


----------------------------- 下載本機 -----------------------------

拉取（pull）曾提交上去的版本
git pull [url] [branch_name]

複製遠端倉儲（repository）的完整鏡像版本
git clone [url]

----------------------------- 使用流程 -----------------------------

SAVE AS
git add .
git commit -m 'dev1_11505051240'



通常流程
建立資料庫 >> 整理完資料夾  >> SAVE AS  

建立分支 >> GO分支 >> 修改資料夾 >>SAVE AS >> GO分支 >> 修改資料夾 >>SAVE AS

GO分支 >> GO分支 >> GO分支 >>
